How to add your Canva embeds

Prefer a guided form? Use the Add Design uploader to submit directly to designs.json or use manual copy mode.

  1. Open script.js.
  2. Add a new object in the designs array.
  3. Paste your Canva embed URL in embedUrl.
  4. Optional: set accent and category for filtering and style.
